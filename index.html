Automation Workflows

Client Onboarding (n8n + QuickBooks + Airtable + Slack)

This workflow automates the client onboarding process end-to-end:

  • Triggered by HubSpot Webhook on new lead submission
  • Creates a QuickBooks Customer record
  • Generates a client folder in Google Drive
  • Logs details into Airtable (“Client Onboarding Log”)
  • Sends notification to Slack #all-demo channel
  • Error handling: Sends Slack alerts + logs errors in Airtable

Tech Stack: n8n, QuickBooks API, Google Drive API, Airtable API, Slack API

AI Invoice Workflow (Power Automate + AI Builder)

This workflow automates invoice extraction and data processing from client emails.

  • Triggered when a new email arrives in the shared client inbox
  • Checks if the email contains a valid invoice PDF attachment
  • Uses AI Builder to analyze and extract structured invoice data:
    • Invoice number
    • Invoice date
    • Vendor name
    • Total amount
    • Line item details
  • Validates extracted data for completeness and consistency
  • Pushes parsed invoice data into the Data Warehouse for reporting and financial reconciliation
  • Logs each processed file and result for audit and tracking

Tech Stack: Microsoft Power Automate, AI Builder, Outlook, SQL Data Warehouse, Power BI

AI Email Task Extractor (Power Automate + AI Builder + Microsoft Teams)

This workflow uses AI to identify and extract actionable tasks from incoming emails and automatically notify the team.

  • Triggered when a new email arrives in the shared or monitored inbox
  • AI Builder model analyzes the email content to determine if it contains a task
  • If the email is a task:
    • Extracts Task Name and Task Category
    • Detects Target Date if mentioned
    • Generates a concise Task Description based on the email body
    • Logs all extracted task details into the shared Task List File (Excel, SharePoint, or Data Warehouse)
    • Sends a notification to Microsoft Teams announcing the new task
  • If the email does not contain a task, the workflow ignores it and performs no action
  • Includes error handling to log any failed extractions or Teams notifications

Tech Stack: Microsoft Power Automate, AI Builder, Outlook, Microsoft Teams, Excel / SharePoint

FB Scheduled Post (n8n + Gmail + Google Sheets + Facebook Graph API)

This workflow automates the process of posting approved social media content to a Facebook Page after email approval.

  • Triggered when a content approval email is received in Gmail with a specific subject
  • Parses the email content and extracts a Post Token to identify the related record in Google Sheets
  • Checks if the email body contains the keyword “approved”
  • If approved:
    • Fetches the corresponding post details (message, image URL, post date) from the Google Sheet
    • Schedules an image post to the linked Facebook Page using the Facebook Graph API
    • Updates the post’s Status column in Google Sheets to “Approved”
  • If the post is not approved, the workflow stops and performs no action
  • Maintains complete audit tracking within Google Sheets

Tech Stack: n8n, Gmail API, Google Sheets API, Facebook Graph API

AI Social Media Content Creator (n8n + Google Gemini + OpenWeather + Google Sheets + Facebook Graph API)

This workflow automatically generates and posts daily AI-created social media content based on current weather data and inspirational quotes.

  • Runs automatically every morning using a Schedule Trigger
  • AI Agent (Google Gemini) generates a unique 5-word inspirational quote each day, ensuring it’s never repeated or too similar to previous entries
  • Stores each new quote in Google Sheets with the date and word count
  • Fetches real-time weather data from OpenWeatherMap for the target location (e.g., Manila, PH)
  • Uses AI to:
    • Retrieve or generate a weather-relevant background image
    • Create a 17-word weather forecast caption suitable for social posting
  • Combines the quote, weather info, and image into a composite post design using the Imejis Design API
  • Automatically uploads the generated image and caption to a Facebook Page via the Facebook Graph API
  • Logs every generated quote and image reference in Google Sheets for tracking and reuse prevention

Tech Stack: n8n, Google Gemini (AI), OpenWeatherMap API, Google Sheets API, Imejis API, Facebook Graph API

AI Receptionist & Appointment Scheduling (n8n + Vapi + Google Calendar + Airtable)

A fully automated voice-based appointment scheduling system powered by Vapi AI. Handles real-time availability checking, booking, rescheduling, and cancellations using Google Calendar, while storing structured call and appointment data in Airtable.

  • Exposes n8n webhook endpoints for: /get-slots, /book-slots, /update-slots, /cancel-slots, and /call-results
  • Generates 30-minute availability slots and formats responses for Vapi tool usage
  • Google Calendar integration: free/busy checks, event creation, updates, and cancellations
  • Airtable integration for logging: caller details, booking metadata, timestamps, event IDs, and workflow results
  • Timezone-aware scheduling (CST / America/Chicago) with date and phrase normalization
  • Error-handling and friendly fallback responses for failed booking actions
  • Designed for natural voice flows using GPT-5 + ElevenLabs voice synthesis

Vapi Configuration:

  • Model: OpenAI GPT-5 (real-time conversational mode)
  • Voice: ElevenLabs — eleven_labs_flash_v2
  • Custom Tools:
    • GetSlots → Calls n8n /get-slots webhook
    • BookSlots → Calls n8n /book-slots webhook
    • UpdateSlots → Calls n8n /update-slots webhook
    • CancelSlots → Calls n8n /cancel-slots webhook

Tech Stack: n8n, Vapi (GPT-5), ElevenLabs, Google Calendar API, Airtable
